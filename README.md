# Stock Analytics Dashboard

A Python application that downloads, analyses and visualises stock market data using Yahoo Finance.

## Features

- Download historical stock data

- Interactive ticker search

- Price visualisation

- Technical indicators (coming soon)

- Portfolio tracking (coming soon)

- AI-powered market summaries (planned)

## Technologies

- Python

- Pandas

- Matplotlib

- yfinance
